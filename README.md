# 실시간 러닝 플랫폼 모여런

> 러닝을 함께 뛰고 싶나요? 어디서든 우린 함께 뛴다! 모여런에서 새로운 러닝을 만나요.

**MoyeoRun**은 실시간으로 같이 달릴 수 있는 모바일 앱 플랫폼입니다. 같은 공간에서 달리지 않아도 앱 서비스를 통해 함께 달릴 수 있도록 지원합니다.

## 서비스 결과물

서비스 주소 : https://moyeorun-web.netlify.app
